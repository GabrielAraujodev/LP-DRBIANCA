### Tendências Identificadas

#### Layout & Estrutura
- **Hero split-screen** (texto à esquerda, visual à direita) — padrão dominante
- **Tipografia oversized** como elemento visual principal
- **Seções com grid assimétrico** — quebra monotonia
- **Cards glassmorphism** — backdrop-filter blur + semi-transparente
- **Scroll reveal animations** — fade-in + translateY
- **Sticky nav compacto** — encolhe ao scroll

#### Visual
- **Dark mode premium** — fundo escuro (#0a0e17, #111827) com acentos dourados/azuis
- **Gradientes sutis** — blur de cores no background (radial-gradient)
- **Espaço em branco generoso** —呼吸感
- **Bordas sutis** — 1px solid rgba para separar elementos
- **Sombras em camadas** — múltiplas elevações para depth

#### Mobile
- **Menu hamburger animado** — transformação em X
- **Cards empilhados** — 1 coluna no mobile
- **Tipografia responsiva** — clamp() para fluid scale

## 4. Próximos Passos

1. **Criar paletas CSS** como arquivos separados em `templates/palettes/`
2. **Criar 3 templates base** com estilos diferentes (dark, light, mixed)
3. **Cada template** com: `index.html`, `style.css`, `assets/`
4. **Variáveis CSS** parametrizáveis para troca fácil de paleta

### Estrutura Proposta para Templates
```

├── dark-premium/       ← fundo escuro, acentos dourados
│   ├── index.html
│   ├── style.css
│   └── assets/

│   ├── index.html
│   ├── style.css
```
